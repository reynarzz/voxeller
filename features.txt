[#Slogan]
The last voxel convert tool you will ever need.

[#Features]
-read ".vox" files from magicaVoxel [Done]
-Export fbx and obj [Done]
-Export convert info as json .txt (num of meshes, colors count, frames, meshes names, etc...)
-Single and multi threaded processing, fully thread safe library.
-Cull faces that a "virtual camera" (manipulated by user) can't see
-If user decides to fully merge all meshes into one, also ask user if wants to remove hidden faces

[#Voxeller lib]
-Libs for Unity, full package (Custom importer for .vox)
-C# bindings
-C++ shared lib (Windows, Macos, Android, IOS, Linux), multiple compilers. (Clang, MSVC, MinGW)

[#Editor]
-Single executable, no dependencies (Release)
-Platforms: Windows, Mac, Linux
